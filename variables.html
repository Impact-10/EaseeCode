Information: Understanding Variables & Data Types

Welcome to the Python world! üêç Let's start by learning about variables and what kind of data we can store in them. Think of variables as boxes where you can store information!

In Python, we have different types of data. Here are a few examples:

# This is an integer (whole number)
x = 10  

# This is a string (text inside quotes)
name = "Python"

# This is a float (a number with a decimal)
pi = 3.14

# This is a boolean (True or False)
is_active = True

Let's break it down!

  • Integers: Numbers like 1, 2, 10, etc.
  • Strings: Words or sentences inside quotes, like "Hello!"
  • Floats: Numbers with decimal points, like 3.14 or 1.5.
  • Booleans: True or False, useful for checking if something is right or wrong.
