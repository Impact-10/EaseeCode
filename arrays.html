Learn Python: Lists and Arrays

In Python, lists allow you to store multiple items in a single variable. Lists are ordered, changeable, and can hold a mix of data types.

1. Defining a List: Lists are created using square brackets [] and can store a variety of data types like integers, strings, or even other lists.

# Example of a list
my_list = [1, 2, 3, 4, 5]
print(my_list)  # Output: [1, 2, 3, 4, 5]

2. Accessing List Elements: You can access elements in a list by using their index. Remember, Python uses zero-based indexing!

# Accessing elements in a list
print(my_list[0])  # Output: 1
print(my_list[3])  # Output: 4

3. Modifying a List: You can change the value of an item in a list by referring to its index.

# Modifying list elements
my_list[2] = 10
print(my_list)  # Output: [1, 2, 10, 4, 5]

4. Adding Items to a List: You can add items to a list using methods like append() and insert().

# Adding items
my_list.append(6)  # Adds 6 to the end
my_list.insert(2, 7)  # Inserts 7 at index 2
print(my_list)  # Output: [1, 2, 7, 10, 4, 5, 6]

5. Removing Items from a List: You can remove items using remove(), pop(), or del.

# Removing items
my_list.remove(10)  # Removes the first occurrence of 10
my_list.pop()  # Removes the last item
del my_list[1]  # Deletes the item at index 1
print(my_list)  # Output: [1, 7, 4, 5]

6. Looping Through Lists: You can loop through a list using a for loop.

# Looping through a list
for item in my_list:
    print(item)
# Output:
# 1
# 7
# 4
# 5

7. List Slicing: You can extract a portion of a list using slicing.

# List slicing
sub_list = my_list[1:3]  # Slices the list from index 1 to 2
print(sub_list)  # Output: [7, 4]

8. Working with Arrays: Arrays in Python are provided by the array module. While lists are more versatile, arrays are more memory-efficient for large data.

# Example of creating an array
import array
arr = array.array('i', [1, 2, 3, 4, 5])
print(arr)  # Output: array('i', [1, 2, 3, 4, 5])
