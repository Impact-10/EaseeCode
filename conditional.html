Learn Python: Conditional Statements

Conditional statements are used to perform different actions based on different conditions. Learn to use the if, else, and elif statements in Python!

1. Simple if statement: An if statement lets you test a condition, and if it's true, the block of code inside the if will run.

# Example of simple if
if number > 10:
    print("Number is greater than 10")

2. if-else statement: An if-else statement lets you test a condition. If the condition is true, one block of code will run; otherwise, another block will run.

# Example of if-else
if number > 10:
    print("Number is greater than 10")
else:
    print("Number is 10 or less")

3. if-elif-else statement: The elif keyword lets you check multiple conditions.

# Example of if-elif-else
if number > 10:
    print("Number is greater than 10")
elif number == 10:
    print("Number is equal to 10")
else:
    print("Number is less than 10")

4. Nested if statements: You can also have if statements inside another if statement, known as a nested if.

# Example of nested if
if number > 5:
    if number < 10:
        print("Number is between 5 and 10")
    else:
        print("Number is greater than or equal to 10")
else:
    print("Number is 5 or less")
