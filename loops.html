Learn Python: Loops & Repetition

Loops allow you to repeat a block of code multiple times. Letâ€™s work with loops in different scenarios to build a strong understanding of them.

1. For Loop (with range): This loop repeats a block of code a fixed number of times.

# For loop with range
for i in range(5):
    print(i)

2. For Loop (with list): Loops can also iterate over items in a list or array.

# For loop with a list
colors = ["red", "blue", "green"]
for color in colors:
    print(color)

3. While Loop: The while loop repeats as long as the condition is true.

# While loop example
i = 0
while i < 5:
    print(i)
    i += 1

4. For Loop (with break): The break statement can be used to exit a loop early.

# For loop with break
for i in range(10):
    if i == 5:
        break
    print(i)

5. For Loop (with continue): The continue statement can be used to skip the current iteration and proceed to the next one.

# For loop with continue
for i in range(5):
    if i == 2:
        continue
    print(i)
